# ⏱ React Stopwatch App

A simple, responsive, and user-friendly stopwatch built using **React JS**.  
This project demonstrates core React concepts like state management, event handling, and component-based architecture.

---

## 🚀 Features

✅ Start the timer  
✅ Stop / pause the timer  
✅ Reset timer to zero  
✅ Time displayed in **MM:SS format**  
✅ Responsive design for all screen sizes  
✅ Clean and modern UI
